# odin-recipes

My first HTML-based project from The Odin Project course.

## What is it?

Just a simple website with few recipes. Home page with recipes and with links to every each. Every single recipe based on it's own page with a description, ingredients, and step-by-step instructions.

### What I'm practicing

- HTML structure (boilerplate, semantic tags)
- Headings of different levels
- Lists (ul,ol)
- Links (absolute and relative)
- Images with alt attributes
- File organization in the project
- Git workflow with meaningful commit messages based
